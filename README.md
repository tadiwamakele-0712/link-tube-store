# Joker Di Genius — Link Tube Store

YouTube music & video link page for **Joker Di Genius**.

## Live site

https://tadiwamakele-0712.github.io/link-tube-store/

## Open locally

```bash
npx serve link-tube-store
```

Then open the URL shown in the terminal.

## Refresh YouTube videos

```bash
cd link-tube-store
node fetch-videos.mjs
```

Commit and push `videos.json` after running the script.
